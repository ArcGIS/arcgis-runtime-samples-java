# Display a Map

Display a map with some imagery.

![](DisplayMap.png)

## How it works

To display a `ArcGISMap`:

1. Create an ArcGIS map using a default `Basemap` such use `Basemap.createImagery()`.
2. Set the map to the map view, `MapView.setMap()`.

## Relevant API

* ArcGISMap
* Basemap
* MapView
